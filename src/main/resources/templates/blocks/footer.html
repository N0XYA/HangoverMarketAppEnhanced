about this project HangoverMarket - магазин алкогольной продукции, после которой не бывает похмелья!

Проект был спроектирован и создан для освоения фреймворка spring-boot, основ HTML и JavaSript.

Роли в проекте:
Кирилл Валиков - создание логики взаимодействия сервиса с базой данных (PostgreSQL). Единолично разобрался и имплементировал первичную логику RESTFul архитектуры, после чего объяснил принципы работы другим участникам проекта. Единолично имплементировал spring security для возможности регистрации и авторизации на сайте. Создал логику отображения отдельной страницы товара при нажатии на товар. (Допиши тут сам потом).

Кирилл Гологузов - создание и имплементация логики взаимодействия frontend с backend, а именно:

- Сохранение изображения в локальном хранилище, на котором развернут backend.
